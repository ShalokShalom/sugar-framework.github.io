Sugar.Router

Sugar.Router defines an alternate format for Plug.Router routing. Supports all HTTP methods that Plug.Router supports, currently get, post, put, patch, delete, and options.

Example

defmodule Routes do
  use Sugar.Router

  get "/", Main, :index
  get "/pages/:slug", Page, :show
end
